# Shopping Cart App

Bevásárlókosár projekt C#-ban, unit tesztekkel. 

**Mik lettek megcsinálva:**
- Kosár logika és osztályok megírva (`BasketItem`, `ShoppingBasket`, `PriceReduction`).
- Szabályok lekezelve (pl. egyforma terméknél csak a darabszám nő, a kedvezmény meg nem viheti mínuszba az árat).
- MSTest unit tesztek megírva az edge case-ekre is.
- Az összes teszt zöld. A bizonyíték képek a `Pictures` mappában vannak.
