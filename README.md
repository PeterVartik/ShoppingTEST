# Bevásárlókosár 

## Áttekintés
C# kosár projekt, megírt tesztekkel és implementált logikával.

---

## Fontos szabályok
- `AddItem()` — ha már van ilyen termék, csak a mennyiség nő.
- `GetTotal()` — darab * ár összege.
- `ApplyFixed()` — nem mehet mínuszba a végösszeg.

- `ShoppingCartAppTests/` (tesztek)
